<h3>9.Compare method and constructor?</h3>
<p><strong>Answer:</strong><br />
Constructor: Used to initialize the instance of a class.<br />Method: Used to perform some function or operation.</p>

<h3>10. What is a singleton class?</h3>
<p><strong>Answer:</strong><br />
Singleton class limits the number of objects created for a class to one but gives the flexibility of creating more objects if the situation changes.</p>

<h3>12. What is the different type of access modifiers?</h3>
<p><strong>Answer:</strong><br />
There are four type of access modifiers as given below:-<br />
• Visible to the overall package. No modifier needed.<br />
• Private &#8211; Visible to class only.<br />
• Public – Visible to the world.<br />
• Protected – Visible to package and subclass.</p>

<p><strong>Answer:</strong><br />
